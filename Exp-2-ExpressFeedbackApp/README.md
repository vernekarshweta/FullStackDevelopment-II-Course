Create an Express application that handles customer feedback.

The application should allow customers to submit feedback through a form.
Feedback data will be stored in an in-memory structure.

An administrator should be able to:

- View all feedback submissions
- Edit feedback
- Delete feedback

Key Features:

1. Customer feedback form
2. Admin routes to view feedback
3. Update feedback
4. Delete feedback
