# Node.js + Express + MongoDB Practice Projects

This repository contains my practice projects while learning  
**Node.js, Express.js, and MongoDB** with problem statements.

I am following a step-by-step approach where each folder contains a small backend project based on a problem statement.

For course reference:
https://www.notion.so/Full-Stack-Development-II-Course-Material-30f936ea50e6806cb710d503386d4031


## 📌 Environment Details

Node Version:
v20.19.0

NPM Version:
11.4.2


## 📌 How this repository is structured

Each folder contains:

- Problem statement
- Source code
- Package.json
- Instructions to run

Example:

1.Project-1-FileSystem/

2.Project-2-Express-with-server/

3.Project-3-API-CRUD/




## 📌 How to run any project

Step 1 — Go inside project folder


cd project-folder-name


Step 2 — Install node modules


npm install


Make sure `node_modules` folder is created.

Step 3 — Run project


npm run start



## 📌 Important Note

Before running any project, always run:


npm install


because `node_modules` folder is not uploaded to GitHub.


## 📌 Scripts used

Each project uses:


npm run start


Make sure package.json contains:


"scripts": {
"start": "node server.js"
}
