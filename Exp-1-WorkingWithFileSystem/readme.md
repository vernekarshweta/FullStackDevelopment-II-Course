Build a command-line task manager application that allows the user to add, view, edit, and delete tasks. Store task data in a JSON file and implement asynchronous operations using Node.js.

Key Features:  
i.	Create tasks with a title and description.

ii.	List all tasks stored in the JSON file.

iii. Update a task's status to "completed."

iv.	Delete a task.

v.	Save and load tasks from a JSON file.
